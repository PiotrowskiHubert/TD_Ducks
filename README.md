THERE ARE FEW TYPES OF ENEMIES (SLIMES) THEY HAVE DIFFERENT SPEED, SPRITE, AND [RUN GAME TO FIND OUT]

RUN: 
1. OPEN CONSOLE 
2. GO TO PROJECT LOCATION FROM CONSOLE EX. cd C:\projects\TD_Ducks
